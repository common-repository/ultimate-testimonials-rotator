Ultimate Testimonials Rotator Plugin is a powerful tool that let you to display multiple testimonial grids and sliders on website using shortcodes.

== Description ==

The Ultimate Testimonials Rotator Plugin is a powerful tool that enables you to display multiple testimonial grids and sliders on your website using shortcodes with categories.

The plugin is fully responsive, which means it can adapt to different screen sizes and devices, including desktops, laptops, tablets, and smartphones.
Additionally, it has swipe enabled functionality, allowing users to navigate through the testimonials with ease. The plugin also boasts infinite looping functionality, which means the testimonials can play indefinitely, creating a seamless user experience.

Furthermore, the plugin is fully accessible, with arrow key navigation, autoplay, and other features that make it easy for visitors to navigate and interact with the testimonials.

All these features combined make the Ultimate Testimonials Rotator Plugin an ideal choice for anyone looking to showcase testimonials on their website in an engaging and user-friendly way.

== Features: ==

  - Testimonial Name
  - Testimonial Title
  - Testimonial Description
  - Select Testimonial Delay Time
  - Select Number of Testimonials to each screen view
  - Select weither you want to show buttons or not
  - Enable or disable infinite Loop
  - Enable or disable Autoplay testimonials

== Installation ==

1. Upload the entire ultimate-testimonials-rotator directory to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Generate the shortcode by creatings sliders.
4. Use the shortcodes to embed the testimonials on any page.

== Procedure ==

 - Generate the shortcodes by creating the slides in the menu. 
 - Simply use the shortcode to embed the Testimonials Rotator on any page.
 - To make shortcode simply add new slider and shortcode will be added in the menu , copy it and paste it to the page. 
 - Each Slider have its own settings , by clicking on edit slider you will be redirected to the settings page.
